# Surfin 管理训练营优秀案例库

本项目是一个纯前端的案例库页面，使用 HTML5 + TailwindCSS CDN 构建，适合直接在浏览器中打开浏览。页面支持中英文切换，并包含跨文化案例库和管理者案例库两大入口。
跨文化案例库中除「案例」外，也支持「国家手册（Playbook）」类型内容（例如印尼手册）。

## 快速预览
- 打开 `index.html` 即可在浏览器中查看页面，无需构建或依赖安装。
- 页面默认加载中文，可通过右上角按钮在中英文间切换。

## 页面结构概览
- **首页 (`#view-home`)**：
  - 顶部标签切换「跨文化管理案例库」与「管理者案例库」。
  - 英雄标题与副标题展示当前库的名称与简介。
  - `#case-grid` 动态渲染案例卡片，点击卡片进入对应案例详情。
- **案例概览与正文 (`#view-caseX-summary` 与 `#view-managerX-summary`)**：
  - 顶部返回首页的导航条。
  - 概览页包含案例亮点、情境与冲突、产生原因等模块。
  - 详情页提供完整正文，包含章节标题与补充说明，适合打印或单页阅读。
  - 国家手册使用 `#view-playbookX-summary` 与 `#view-playbookX-article` 视图结构。
- **交互导航**：
  - `navigateTo(viewId)` 控制视图切换，并维护返回栈支持「Esc」键返回上一视图或首页。
  - 语言切换通过 `toggleLang()` 切换 `data-lang` 标记元素的显示文本。

## 样式与布局说明
- 使用 TailwindCSS CDN 快速提供网格、排版与阴影等实用类。
- 自定义样式集中在 `<style>` 中：
  - 玻璃态卡片、暗色与浅色背景适配。
  - 管理者案例正文的阅读友好配色（白底、深色文字、浅色提示卡）。
- 布局多采用 Flex 与 CSS Grid，确保在移动端与桌面端均保持良好展示。
