## 🎨 视觉设计规格

### 玻璃拟态效果

```css
background: rgba(15, 23, 42, 0.7);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.1);
box-shadow: 0 25px 50px -12px rgba(0, 0, 0, 0.5);
```

### 动态背景

- 渐变色流动动画 (Surfin Blue + 深海绿)
- 微妙的粒子/光点漂浮

### 输入框交互

- 聚焦时发光边框
- 输入时微弱呼吸效果
- 错误时红色边框 + 震动

## 🔒 安全实现

### 密码存储

- 使用 SHA-256 哈希存储密码
- 不在前端代码中暴露明文密码

### 记住我功能

- localStorage 存储验证状态
- 7 天自动过期
- 可手动登出
